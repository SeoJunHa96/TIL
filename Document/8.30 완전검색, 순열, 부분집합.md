- 선택정렬

  ```python
  def selectionsort(a):
      n = len(a)
      for i in range(n-1):
          minI = i
          for j in range(i+1, n):
              if A[j] < A[minI]:
                  minI = j
          A[minI], A[i] = A[i], A[minI]
  ```



- 재귀 호출을 통한 순열 생성

  ```python
  def permutations(arr, r):
      def backtrack(start):
          if start == r:
              result.append(arr[:r])
              return
          for i in range(start, len(arr)):
              arr[start], arr[i] = arr[i], arr[start]
              backtrack(start + 1)
              arr[start], arr[i] = arr[i], arr[start]
  
      result = []
      backtrack(0)
      return result
  ```

  

- 비트연산자를 통한 부분집합

  ```python
  arr = [3, 6, 7, 1, 5, 4]
  # n : 원소의 개수
  n = len(arr)
  
  # 모든 부분집합을 저장할 리스트
  subset_list = []
  
  # 이진수와 비트연산자를 활용
  for i in range(1<<n):  # 1<<n == 2**n(부분집합의 개수)
      subset = []
      for j in range(n):  # 부분집합을 만들 원소의 개수
          if i & (1<<j):  # j번 비트가 0이 아니면
              subset.append(arr[j])
      subset_list.append(subset)
      
  print(subset_list)
  ```
