### 8.2 2차원 배열

- 2차원 배열의 선언

  1차원 list를 묶어놓은 list

  2차원 이상의 다차원 list는 차원에 따라 index를 선언

  ```python
  # 2차원 arr = [0][0]
  # 3차원 arr = [0][0][0]
  ```

  2차원 리스트의 선언 : 세로 길이(행의 개수), 가로 길이(열의 개수)를 필요로 함

  python에서는 데이터 초기화를 통해 변수 선언과 초기화가 가능.

  ```python
  arr = [[0, 1, 2, 3], [4, 5, 6, 7]] (2행 4열의 2차원 List)
  """
  0 1 2 3
  4 5 6 7
  """
  
  N = int(input)
  arr = [list(map(int, input().split()))] for _ in range(N)
  """
  3

  7 8 9
  """
  ```

- 배열 순회

  n * m 배열의 n*m 개의 원소를 빠짐없이 조사하는 방법

  - 행 우선 순회


  ```python
  # i 행의 좌표
  # j 열의 좌표
  
  s = 0
  for i in range(n):
      for j in range(m):
          s += array[i][j] # 필요한 연산 수행.
  ```

  - 열 우선 순회


  ```python
  for j in range(m):
      for i in range(n):
          f(arr[i][j])
  ```

  - 지그재그 순회


  ```python
  for i in rnage(n):
      for j in range(m):
          f(arr[i][j+(m-1-2*j)*(i%2)])
          
          
  ```

  - 델타를 이용한 2차 배열 탐색


  ```python
  arr[0...N-1][0...N-1]  # N*M배열
  di [] <- [0, 1, 0, -1]
  dj [] <- [1, 0, -1, 0]
  for i: 0 -> N-1
      for j : 0 -> N-1:
          for k in range(4):
              ni <- i + di[k]
              nj <- j + dj[k]
              if 0 <= ni <N and 0 <=nj<N  # 유효한 인덱스면
                  f(arr[ni][nj])
                  
  ```

  ```python
  arr = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
  
  """
  3
  1 2 3
  4 5 6
  7 8 9
  """
  
  di = [0, 1, 0, -1]
  dj = [1, 0, -1, 0]
  N = int(input())
  arr = [list(map(int, input().split()))for _ in range(N)]
  
  max_v = 0
  for i in range(N):  # 모든 원소 arr[i][j]에 대해
      for j in range(N):
          # arr[i][j]를 중심으로
          s = arr[i][j]
          for k in range(4):
              ni, nj = i + di[k], j + dj[k]
              if 0 <= ni<N and 0 <= nj <N :    #배열을 벗어나지 않으면
                  s += arr[ni][nj]
          # 여기까지 주변 원소를 포함한 합
  
          if max_v < s :
              max_v = s
  
      print(max_v)
  ```

  - 전치 행렬


  ```python
  # i : 행의 좌표, len(arr)
  # j : 열의 좌표, len(arr[0])
  arr = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]  # 3*3 행렬
  
  for i in range(3):
      for j in range(3):
          if i < j :
              arr[i][j], arr[j][i] = arr[j][i], arr[i][j]
  ```

```python
T = 10  # 테스트 케이스의 수를 10으로 가정.

for tc in range(1, T+1):
    t = int(input())
    N = 100
    M = 100
    arr = [list(map(int, input().split())) for i in range(N)]

    max_v = 0

    # 각 행과 열의 합을 구하는 부분을 수정해야 합니다.
    for i in range(N):
        sum_i = 0
        sum_j = 0  # 각 열의 합을 초기화해야 합니다.
        for j in range(M):
            sum_i += arr[i][j]
            sum_j += arr[j][i]  # 각 열의 합을 누적합니다.

        # 각 행과 열의 합 중 최대값을 max_v와 비교하여 갱신합니다.
        max_v = max(max_v, sum_i, sum_j)

    # 오른쪽 대각선의 합을 구하는 부분을 수정해야 합니다.
    sum_R = 0
    for i in range(N):
        sum_R += arr[i][i]

    # 왼쪽 대각선의 합을 구하는 부분을 수정해야 합니다.
    sum_L = 0
    for i in range(N):
        sum_L += arr[i][N-1-i]

    # 오른쪽 대각선과 왼쪽 대각선의 합 중 최대값을 max_v와 비교하여 갱신합니다.
    max_v = max(max_v, sum_R, sum_L)

    # 각 테스트 케이스마다 최대값을 출력합니다.
    print(f'#{tc} {max_v}')

```
